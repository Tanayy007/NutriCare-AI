# 🥗 NutriCare AI: Hospital Meal Optimization System

**NutriCare AI** is a professional, AI-powered nutritional command center designed for modern hospitals. It generates personalized, clinically-safe diet plans for patients based on their medical conditions, vitals, and dietary preferences.

---

## 🚀 Live Features

### 🧠 Intelligent Optimization
Our algorithm analyzes patient data against clinical rules to ensure every meal is safe and effective for recovery.
- **Diabetes**: Low sugar, high fiber focus.
- **Hypertension**: Low sodium, high potassium rules.
- **Heart Disease**: Low saturated fats, heart-healthy Omega-3s.
- **Anemia**: Iron-rich pairings with Vitamin C.

### 📊 Real-Time Analytics
- **Nutrient Breakdown**: Visual progress bars showing Calories, Protein, Carbs, and Fats relative to the patient's daily target.
- **Dynamic Risk Alerts**: Instant clinical warnings (e.g., "⚠️ Avoid high sodium") based on selected conditions.
- **Smart Swapping**: Don't like a meal? Click "Alternative" to instantly generate another safe, AI-vetted replacement.

### 🧪 Modern Medical UI
- **Premium Aesthetics**: Soft teal/blue medical theme with glassmorphism and smooth micro-animations.
- **Mobile Responsive**: Fully optimized for tablets and mobile devices for bedside use by hospital staff.

---

## 🛠️ Tech Stack

- **Frontend**: Vanilla HTML5, CSS3, JavaScript (ES6+).
- **Icons**: Lucide Icons.
- **Typography**: Outfit & Inter (Google Fonts).
- **Architecture**: Pure Frontend (No backend required for prototype).

## 🏃 How to Run Locally

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/YOUR_USERNAME/NutriCare-Ai.git
    ```
2.  **Open the App**:
    - Simply open `index.html` in any modern web browser.
    - *Recommended*: Use a local server like [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) in VS Code or `http-server`:
      ```bash
      npx http-server ./
      ```
